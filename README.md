# Shoply

Shoply — frontend React shopping site (demo).

## Features included
- Homepage with hero banner, logo and favicon
- Categories: Clothing, Gadgets, Sports, Housing & Kitchen
- 12 products per category (realistic names, ₦ prices, external stock images)
- Add-to-cart with delivery fee logic (₦1500 if item price < ₦5000)
- Cart/Checkout page showing subtotal, delivery, and total
- Chatbot with delivery info and contacts
- Search bar and footer

## Run locally
1. Extract the ZIP and open the `shoply_full_project` folder in terminal.
2. Run:
```bash
npm install
npm start
```
3. Open http://localhost:3000

## Deploy
Push the folder to GitHub and deploy with GitHub Pages or Vercel/Netlify.
